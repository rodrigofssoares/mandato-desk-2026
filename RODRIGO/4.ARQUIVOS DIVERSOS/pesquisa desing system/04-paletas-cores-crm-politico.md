# Paletas de Cores para CRM Politico/Institucional

Pesquisa realizada em 07/04/2026 para evolucao do Mandato Desk 2026.

---

## Psicologia de Cores para Contexto Politico

- Usuarios julgam confiabilidade em **50ms**
- Interfaces dark blue sao classificadas como **23% mais confiaveis**
- Navy transmite autoridade e expertise intelectual — ideal para gabinete politico
- Cor sozinha nao constroi confianca, mas cor errada a destroi

---

## Sistemas de Design Governamentais de Referencia

### USWDS (U.S. Web Design System)
- **URL:** https://designsystem.digital.gov
- Primary Blue: `#005ea2`
- Tokens de tema e estado completos
- Referencia mundial para design governamental

### GOV.UK Design System
- **URL:** https://design-system.service.gov.uk
- Brand Blue: `#1d70b8`
- Abordagem minimalista com foco maximo em acessibilidade

### GOV.BR (Brasil)
- **URL:** https://www.gov.br/ds/
- Escala Blue: `#eff6fb` (blue-5) a `#11181d` (blue-90)
- Cores semanticas inclusas
- Referencia direta para CRM de gabinete brasileiro

### UAE Design System
- Gold accent: `#ca9a2c` para transmitir confianca
- Insight: "cor sozinha nao constroi confianca" — precisa de consistencia e contexto

## Comparacao com Paleta Atual

O Mandato Desk 2026 atualmente usa:
- **Primary:** Sky Blue (`#0ea5e9` principal, escala 50-900)
- **Dark bg:** `#020617` / Cards: `#0f172a` / Borders: `#1e293b`
- **OKLCH:** Sim, ja implementado

### Analise:

| Aspecto | Atual (Sky Blue) | Paleta A (Navy) | Paleta B (Slate) | Paleta C (Dark) |
|---------|-----------------|-----------------|------------------|-----------------|
| Confianca | Media | **Alta** | Media-Alta | Media |
| Modernidade | Alta | Media | Alta | **Muito Alta** |
| Seriedade | Media | **Alta** | Alta | Media-Alta |
| Acessibilidade | Boa | **Excelente** | Boa | Boa |
| Diferenciacao | Baixa | **Alta** | Media | Alta |

### Recomendacao:
**Paleta A (Navy Institucional)** para o contexto politico do Mandato Desk, com elementos da Paleta C para dark mode. O navy transmite mais autoridade que sky blue e se alinha com padroes governamentais (USWDS, GOV.BR).
