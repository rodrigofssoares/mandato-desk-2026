## Fontes Analisadas

### 1. Inter

| Atributo | Detalhe |
|----------|---------|
| **URL** | https://fonts.google.com/specimen/Inter |
| **Designer** | Rasmus Andersson |
| **Licenca** | SIL Open Font License (free comercial) |
| **Pesos** | Variable: 100-900 (Thin a Black) |
| **Quem usa** | Notion, Linear, Shopify, GitHub, Mozilla, Figma (antes) |
| **Melhores pesos UI** | 400 (body), 500 (labels/botoes), 600 (subheadings), 700 (headings) |

**Pros:**
- Maior legibilidade entre sans-serifs gratuitas em tamanhos pequenos
- Variable font = arquivo unico, todos pesos (loading mais rapido, ate 3x)
- OpenType features extensos (tabular numbers, case alternates)
- 414 bilhoes de requests Google Fonts em 2024-2025

**Contras:**
- Problema de uso excessivo: "Quando produto, concorrente e templates usam Inter, tipografia para de ser diferencial"
- 57% crescimento YoY — virando o novo Arial/Helvetica
- Empresas lideres (Figma, Vercel) ja migraram para fontes custom

**Status no projeto:** JA UTILIZADA como `font-sans`

## Comparacao com Setup Atual

O Mandato Desk 2026 atualmente usa:
- **Inter** (font-sans) — corpo, labels, botoes
- **Space Grotesk** (font-display) — titulos, headings
- **Cinzel** (font-serif) — destaques especiais

### Analise:
- **Inter** — Excelente escolha, mas esta se tornando "generica". Considerar trocar para **Manrope** ou **DM Sans** para diferenciar
- **Space Grotesk** — Boa fonte display, mas **General Sans** ou **Cabinet Grotesk** trariam mais personalidade institucional
- **Cinzel** — Serif classica, adequada para destaques. Alternativa mais moderna: **DM Serif Display**

### Recomendacao:
Manter **Inter** (comprovada, performatica) ou migrar para **Manrope** (melhor design de numerais para CRM). Para headings, considerar **General Sans** (mais institucional que Space Grotesk).
