Três caminhos visuais para a integração WhatsApp

O MVP é o mesmo nas três versões: 2 contas Z‑API, texto, recebimento, status e tela WhatsApp Web com 4 subabas. Muda o caráter da interface — vibe, paleta, tipografia, densidade de informação.

Todas as 3 já com tabs funcionando · clique e navegue
— Versão 01

Verdana Eleitoral

"Memo de gabinete brasileiro encontra WhatsApp Web"

VibeQuente, cordial, gabinete físico digitalizado
PaletaVerde‑musgo · creme · terracota · ouro
FontsFraunces (serif) + Atkinson Hyperlegible
DensidadeConfortável, espaçosa
  • Familiar pra quem usa WhatsApp Web — adoção rápida
  • Caráter brasileiro político (textura papel, tipografia editorial)
  • Toque artesanal — não parece "dashboard genérico"
  • Pode parecer "menos sério" pra quem espera CRM corporativo
  • Menos densidade de informação por tela
Abrir V1
— Versão 02

Navy Institucional

"Câmara dos deputados encontra workspace premium"

VibeSóbrio, refinado, escritório institucional
PaletaNavy profundo · creme · dourado fino
FontsPlayfair Display (serif) + Manrope
DensidadeEquilibrada, hierarquia forte
  • Alinhado ao DS Mandato (Navy Institucional já decidido em memória)
  • Tom "advocacia premium" — passa autoridade política
  • Tipografia editorial, ornamentos sutis (numeração romana, dividers de ouro)
  • Menos calor humano que V1 — mais formal
  • Pode parecer "pesado" se usado o dia inteiro
Abrir V2
— Versão 03

Híbrido Moderno

"Linear / Vercel — power-user dark, command palette, ⌘K"

VibeProdutividade densa, dark mode, atalhos
PaletaNear-black + neon teal + violeta
FontsOutfit + JetBrains Mono
DensidadeAlta — máximo de info na tela
  • Atendentes operam o dia todo — dark mode reduz fadiga visual
  • Atalhos teclado (⌘K, ⌘1..⌘4) aceleram quem pegar o jeito
  • Painel de contexto à direita: ficha, atalhos, atividade
  • Visual "engenharia/SaaS moderno"
  • Mais distante do "WhatsApp clássico" — curva de aprendizado
  • Dark mode pode não agradar usuários acostumados ao light
  • Mais elementos = mais código de manutenção
